## Demo Authentication

The activities app now includes token-based authentication with role checks for mutating actions.

- Public users can still view activities and participant rosters.
- Only `admin` and `staff` users can sign students up or unregister them.
- Passwords are stored as PBKDF2 hashes in `src/users.json` for this demo.
